БОЙОВИЙ ПОСІБНИК

🔐 1. ВХІД У СИСТЕМУ

Натисни кнопку ЗАПУСК.

📍 2. МАРШРУТ / МАПА

Як поставити ціль: Двічі швидко тапни по карті.
Я ТУТ (БЕЗ GPS): Використовуй під РЕБом. Натисни і ткни пальцем у карту туди, де ти зараз.

🧭 3. КОМПАС / ПОВОДИР

Крокомір: Якщо немає GPS, увімкни його і він сам рахуватиме кроки.
Поводир (Вібро): Телефон вібрує, коли дивишся точно на ціль.

🌌 4. АСТРО / БЕЗ GPS

Коли компас збився: Тримаєш телефон рівно -> 'КАЛІБРУВАТИ ГОРИЗОНТ' -> піднімаєш у небо.

👁 5. ОПТИКА / ШІ

ІЧ-ФІЛЬТР: Робить картинку контрастною для сутінків.
ШІ SCAN: Камера сама шукає людей або техніку.

💬 6. ТЕКСТ / QR-РАЦІЯ

Пишеш текст -> 'Створити QR'. Напарник відкриває програму і сканує.

🛡 7. ОХОРОНА / ЩИТ

Натискаєш 'АКТИВУВАТИ ЗАХИСТ' і кладеш телефон. Вібрація = тривога.

🆘 8. SOS / МАЯК ВИЖИВАННЯ

Натискаєш ВЕЛИКУ кнопку SOS. Екран яскраво блимає, лунає звук.
